# Profile Image Placeholder

Place your profile picture here and name it `profile.jpg`.

The hero section on the home page will display this image.

## Recommended specifications:
- **Format:** JPG or PNG
- **Dimensions:** Minimum 800x1000 pixels (portrait orientation)
- **Aspect Ratio:** Approximately 4:5 (width:height)
- **Quality:** High resolution for best results
- **Subject:** Professional headshot or upper body photo
- **Background:** Clean or blurred background works best

## How to add your photo:
1. Rename your photo to `profile.jpg` (or `profile.png`)
2. Place it in this folder (`public/images/`)
3. The image will automatically appear on the home page hero section

**Note:** If using PNG format, update the image path in `src/app/page.tsx` from `profile.jpg` to `profile.png`.
